1、什么是数据库？
	数据库就是存储数据的仓库，其本质是一个文件系统。数据按照特定的格式将数据存储起来。
	用户可以对数据库中的数据进行增加，修改，删除以及查询操作。

2、什么是数据库管理系统
	DBMS 是一种操作和管理数据库的大型软件，用于建立、使用和维护数据库，
	对数据库进行统一的管理和控制，以保证数据库的安全性和完整性。
	用户通过数据库管理系统访问数据库中的数据。
	
3、数据库表
	数据库中以表为单位组织单位存储数据。

4、表数据
	根据表字段所规定的数据类型，我们可以向其中填入一条条的数据。
	每条数据类似类的实例对象，表中的一行一行的信息，我们称之为记录
	
5、SQL 语句
	DDL:
	数据定义语言，用来定义数据库对象，如create，alter，drop等
	DML:
	数据操作语言, 用来对表中的数据进行更新insert, delete,update等
	DCL：
	数据控制语言，用来定义数据库的访问权限和安全级别，以及创建用户
	DQL：
	数据查询语言，用来查询数据库中表的记录，如select，from，where等
	
6、Sql通用语法
	1、SQL语句可以单行或者多行书写，一分号结尾
	2、可使用空格和缩进来增强语句的可读性
	3、大小写取决于根据当前的操作系统(为了保持一致性，默认区分大小写) windows系统不区分大小写，Linux系统区分大小写
	4、同样可以使用/**/来表示注释信息
	
	
7、标识符命名规则：
	大小写取决于当前的操作系统，标识符的字符可以使用任意字符，中文，数字，中文等，
	但一些特殊组合，如纯数字，特殊符号，包括mysql的关键字等，要使用反引号包裹才可以使用
	中文可以使用，但要注意编码问题
	
8、 每创建一个数据库，都会在MySql的data目录下生成一个与数据库名一样的文件夹
		windows 下 默认在c:\program data\mysql \data
