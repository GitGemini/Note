/*构造函数
 * 构建创造对象时调用的函数
 * 事物一旦产生就具有的属性要在构造函数中进行初始化
 * 作用：
 * 1、初始化对象
 * 2、
 * 特点:
 * 1、一个类中如果没有定义过构造函数，则该类中会有一个默认的空的构造函数
 * 2、如果在类中定义了指定的构造函数，那么类中默认的构造函数就没有了
 * 
 * 与一般函数的区别:
 * 1、在对象创建时就会调用与之对应的构造函数，进行初始化
 * 2、一般函数在对象创建后，需要函数的功能时才调用
 * 3、构造函数对象创建时值调用一次，一般函数在创建后可以被调用多次
 * 
 * 对象初始化时，首先在堆中开辟内存空间，初始化静态属性，静态方法，私有属性，私有方法，静态代码块，构造函数
 * 然后构造函数进栈，执行完毕后弹栈，然后返回对象的引用地址，赋值给变量
 */
