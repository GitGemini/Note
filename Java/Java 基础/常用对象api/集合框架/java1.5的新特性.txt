/**
 * 1.5新特性
 * 
 * 1、forEach
 * for(x : c){
 * 		
 * } 
 *
 * for与forEach的区别
 *	
 * for可以对语句执行很多次，因为可以自定义控制循环的增量和条件 
 * 
 * forEach是一种简化形式，他必须有被遍历的目标，该目标要么是数组要么是Collection集合，只能遍历不能对集合进行增删改操作
 * 
 * 如果要对集合进行增删该改操作用for或者迭代器   (尤其是ListIterator ，可以增删)
 * 
 * 2、函数可变参数
 * 注意：可变参数必须放在参数列表最后
 *  public static int add(int... m)//其实就是一个数组，但是接收的是数组中的元素，编译时自动将这些元素封装数组，简化了调用者的书写
 *	{
 *		return 0;
 *	}
 *  
 *  3、静态导包
 *  其实是导入类的静态成员
 *  import static java.util.Collections.sort;   导入一个类中的静态方法，调用时就不用写类名.方法(),可以直接写方法();
 */
